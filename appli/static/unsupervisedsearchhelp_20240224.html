Similarity Search Help
Similarity search function will find, in EcoTaxa, objects which are visually similar to a given (called 'seed') one.
Each search result materializes as an added sort order option. When this option is selected, visually similar objects will be displayed near the seed.
This specialized and dynamic order is only available when a Prediction was executed on the project since last import.

Notes:
  • Searches are not saved in your preferences, they vanish if the page is reloaded.
  • Like any other sort order, similarity search can be sent via 'Share Page' option.
  • If sort is descending , the 'seed' appears as the first (top left) image of the result, with sign.
  • If sort is ascending , the 'seed' appears as the last (bottom right) image, in last page of the result.
  • If filtered, the 'seed' does not appear in the result, but similarity order is still active. It can be useful in some situations.
