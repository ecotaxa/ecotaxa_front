To create/import/delete additional categories such as nauplii, like, female… you can either:
  • Edit categories in the application
  • Create a table to import/delete multiple categories
Example of table: table_import.txt Format of table:
ID_SourceParent ID_Source NameStatusType
67815m104like1morpho
46068m130nauplii1morpho
69363m138bell0morpho
ID_SourceParent : id of the parent from the UniEuk taxonomy (or any other Id), for example M104 (like) is child of Cnidaria (Id 67815)
ID_Source : your givent Id
Name : the given name for the new category (this name can be changed if necessary)
Status :
  • 1 : to create or keep active
  • 0 : to remove
Type : free text (not utilized by Ecotaxa)

NOTE : « _ » et « + » are automatically replaced by « »
